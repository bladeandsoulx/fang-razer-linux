## Approved Release Text

````markdown
# Fang v0.9.4 — One-command Linux installer

Installing Fang is now as simple as opening Terminal and pasting one line:

```bash
curl -fsSL https://github.com/bladeandsoulx/fang-razer-linux/releases/latest/download/install.sh | bash
```

Run it as your normal desktop user—do not add `sudo`. The installer asks for
your password only when it is ready to install the verified packages.

## What’s new

- One-command installation and upgrades for supported x86_64 Linux systems.
- Support for Ubuntu 22.04/24.04, Debian 12/13, and Fedora 43/44.
- Compatible Ubuntu, Debian, and Fedora derivatives are detected automatically
  and shown a compatibility notice.
- Checksums, package metadata, app/daemon versions, and downgrade safety are
  checked before administrator access is requested.
- Fang and `fangd` are installed together, the service is enabled, and your user
  is given access when needed.
- Releases now publish as an immutable six-file set: installer, checksums, two
  DEBs, and two RPMs.
- The Support screen now clearly lists BNB Smart Chain (BEP20) and Ethereum
  (ERC20) as accepted USDT networks.

If the installer says group access was added, log out and back in once before
opening Fang.

**Full Changelog:** https://github.com/bladeandsoulx/fang-razer-linux/compare/v0.9.3...v0.9.4
````
